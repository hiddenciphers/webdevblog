I was introduced to the Flexbox Froggy game which I loved! It was super fun to learn about Flexbox layout using this awesome site.


I then learnt about Flex Sizing (shrinking & growing) by using the properties:

  • flex-basis:

    • Determines the width of flex items in row or height of flex items in column


  • min-width:

    • Determines the minimum size the flex items can shrink to


  • max-width:

    • Determines the maximum size the flex itms can grow to


    When min-width is larger than flex-basis, the flex-basis is ignored & min-width is applied.


  • flex-grow: 1;

    • Turning on flex-grow: allows items to grow past the flex-basis: to take up the full row evenly


  • flex-shrink: 1;

    • Turning on flex-shrink: sets items to size of flex-basis: by default. The flex items will not grow but shrink until it reaches min-width:. If min-width: is not set the flex items will shrink to the minimum width of the content.
