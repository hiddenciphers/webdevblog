HTML (Hyper Text Markup Language) is the language Web Developers use to structure basic one-page & multi-page websites using HTML tags to contruct HTML elements which can then be styled using CSS (Cascading Style Sheets) which is an accompanying language used to target the HTML elements & style them with the use of CSS properties which consist of many options such as:

  • Color
  • Background Color
  • Font-Weight
  • Font-Size
  • Font-Family
Just to name a few...


Most importantly, I learnt about the "Box Model" which consists of 3 properties.

  1. Border

    2.
    • The 'border' property is used to define the border around an element.
    • It consists of three sub-properties: 'border-width', 'border-style', & 'border-color'.

  2. Margin

    3.
    • The 'margin' property controls the space outside an element's border.
    • It can be set as a single value (applies to all sides) or as four values (for each side individually)
    • Example: 'margin: 10px;' creates a 10-pixel margin on all sides.

  3. Padding

    4.
    • The 'padding' property controls the space between an element's content & its border.
    • Similar to margin, it can be set as a single value or as four values.
    • Example: 'padding: 15px;' adds a 15-pixel padding to all sides.
Understanding & effectively using these properties is crucial for Web Development, as they play a significant role in creating the layout & appearance of web pages.
